קונספט 1 / 5 דקות

שיחה רגילה vs. Project

בשיחה רגילה — קלוד לא זוכר כלום מהפעם הקודמת. כל שיחה מתחילה מאפס.

ב-Project — יש לו זיכרון קבוע. הוא יודע מי אתם, מה הסגנון שלכם, ומה החומרים שהכנסתם לו.

📁

System Prompt

ההוראות הקבועות לקלוד — "אתה עוזר שיווקי של [עסק], תמיד כתוב בעברית, הטון הוא מקצועי אך חם..."

📚

Knowledge Base

מסמכים שקלוד יכול לקרוא — קטלוג מוצרים, מחירון, מדיניות החזרות, היסטוריית לקוחות.

💬

שיחות שמורות

כל השיחות בתוך Project מאורגנות ביחד — קל למצוא, קל להמשיך.

קונספט 2 / 5 דקות

System Prompt — הנשק הסודי

ה-System Prompt הוא ההוראה שקלוד מקבל לפני כל שיחה. הוא לא רואה אותו המשתמש — רק קלוד.

דוגמה לSystem Prompt מנצח
אתה [שם], עוזר אישי מקצועי של [שם העסק].

על העסק: [תיאור קצר]
קהל היעד: [מי הלקוחות]
טון הדיבור: מקצועי, חם, ישיר. לא פורמלי מדי.
שפה: תמיד עברית, אלא אם ביקשו אחרת.
אסור: להבטיח דברים שלא אושרו, לדבר על מתחרים.

כשאתה לא בטוח — שאל לפני שאתה עונה.
קונספט 3 / 5 דקות

Knowledge Base — להאכיל את קלוד

קלוד יכול לקרוא קבצים שמעלים לתוך ה-Project: PDF, Word, טקסט רגיל.

💡
מה כדאי להכניס? מחירון, שאלות נפוצות של לקוחות, סגנון כתיבה לדוגמה, מדיניות עסקית. ככל שיש לו יותר — הוא מדויק יותר.
⚠️
מגבלה חשובה: Knowledge Base לא מחליף זיכרון — קלוד קורא את הקבצים בכל שיחה מחדש. כמות הקבצים מוגבלת לפי ה-Context Window.

דמו 2 / ~9 דקות — השוואה

עם Project vs. בלי Project

תראה את אותה שאלה בדיוק — פעם בשיחה רגילה, פעם בתוך ה-Project.

שאלת הבדיקה
כתוב לי מייל follow-up ללקוח שביקש הצעת מחיר לפני שבועיים ולא חזר אלינו.
👀
מה תראו: בשיחה רגילה — מייל גנרי. ב-Project — מייל בסגנון העסק, עם השם הנכון, הטון הנכון, ומידע מהקבצים שהעלינו.
דמו 3 / ~8 דקות — Knowledge Base בפעולה

שאלות שקלוד עונה מתוך המסמכים

Prompt שמשתמש ב-Knowledge Base
לקוח שואל: "כמה עולה שירות X? ויש לכם אפשרות תשלומים?"

ענה לו בשם העסק, בהתאם למידע שיש לך.

👉 קלוד יענה על פי המחירון שהעלינו — לא יומצא מידע.

סיכום המפגש

מה לקחנו הביתה

📌 3 דברים לזכור
1

System Prompt = זהות של קלוד

ככל שתהיה יותר ספציפי — קלוד יתנהג יותר כמו חבר שמכיר את העסק שלך, לא כמו זר.

2

Knowledge Base = זיכרון הידע

העלה כל מסמך שיש לך על העסק. קלוד יקרא הכל לפני שיענה.

3

Project = השקעה חד פעמית

כמה שעות של הגדרה — ואז כל שיחה עתידית מתחילה מרמה גבוהה יותר.

🏠 שיעורי בית
  • השלם את ה-Knowledge Base — עוד 2-3 מסמכים
  • השתמש ב-Project לפחות 3 פעמים השבוע
  • שפר את ה-System Prompt בהתאם לתוצאות
